# AI Cybersecurity Diagnostic Assitant
A Python-based all-in-one cybersecurity diagnostic tool that helps analyze system health, detect suspicious activity, scan network ports, and troubleshoot system issues using a smart rule-based engine.

This project was created for situations where a laptop starts malfunctioning, slowing down, or behaving suspiciously, but immediate technical support is not available. Instead of waiting for a service technician, users can quickly analyze their system, understand the problem, and follow simple steps to fix it themselves.

The tool also helps users monitor the overall health and security of their device to prevent future problems.

Unlike many AI tools that overwhelm users with confusing recommendations or incomplete troubleshooting steps, this assistant provides:
	• Clear and simple solutions 
	• Step-by-step troubleshooting 
	• Root cause explanations 
	• Prevention strategies for future issues 
It acts like a lightweight offline Security Operations Center (SOC) assistant for personal cybersecurity analysis and learning.
Features
1. System Monitoring
	• Fetch system information 
	• Analyze system performance 
	• Monitor CPU-intensive processes 
	• Check startup applications 
2. Port Scanner
	• Scan common attack-prone ports 
	• Perform full port scan (1–1024) 
	• Check specific port status 
	• Detect suspicious open ports 
3. Malware Analysis
	• Detect suspicious processes using CPU spikes and keywords 
	• Identify unusual network connections 
	• Analyze startup entries 
	• Generate risk levels (Low / Medium / High) 
4. Smart Issue Diagnostic System
Accepts user-described system problems and identifies:
	• Network issues 
	• Performance issues 
	• Malware infections 
	• Hardware failures 
Provides:
	• Root cause analysis 
	• Step-by-step fixes 
	• Prevention methods 
	• Easy troubleshooting roadmap 
5. Full System Security Check
	• Complete system analysis 
	• Common port scan integration 
	• Malware behaviour analysis 
	• Overall system risk evaluation 

Technologies Used
	• Python 3.x 
	• psutil for system monitoring 
	• socket for network and port scanning 
	• os module for system interaction
